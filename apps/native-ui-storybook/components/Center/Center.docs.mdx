# Center

When you need to center-align content, the Center component comes in handy. It is a layout component that can be used with other components to create complex layouts and positioning.

## Usage

> This component should be wrapped in a `NativeUIProvider` component to ensure that the correct theme is applied.

<Canvas>
  <NativeUIProvider>
    <Center>
      <Center bg="$cyan500" h={200} w={300}>
        <Text color="$white">This is the center.</Text>
      </Center>
    </Center>
  </NativeUIProvider>
</Canvas>

```jsx
import { Center, Text } from '@utilitywarehouse/native-ui';

const MyComponent = () => (
  <Center bg="$cyan500" h={200} w={300}>
    <Text color="$white">This is the center.</Text>
  </Center>
);
```

For more information, see the [Gluestack Center component documentation](https://gluestack.io/ui/docs/components/layout/center).
