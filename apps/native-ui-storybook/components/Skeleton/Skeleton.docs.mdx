# Skeleton

The `Skeleton` component is used to show a loading state for a component that is still loading.

## Usage

<Primary />

<Controls />

```jsx
import { Center, Skeleton, Text } from '@utilitywarehouse/native-ui';

const MyComponent = () => (
  const [loading, setLoading] = useState(true);

  return (
    <Center>
      {loading ? <Skeleton width={200} height={16} /> : <Text>Loaded</Text>}
    </Center>
  );
);
```

## Props

| Name              | Type     | Default   | Description                           |
| ----------------- | -------- | --------- | ------------------------------------- |
| `width`           | `number` | `-`       | The width of the skeleton.            |
| `height`          | `number` | `-`       | The height of the skeleton.           |
| `backgroundColor` | `string` | `$grey75` | The background color of the skeleton. |
| `borderRadius`    | `number` | `$sm`     | The border radius of the skeleton.    |
