## Props

| Property    | Type                                              | Description                      | Default     |
| ----------- | ------------------------------------------------- | -------------------------------- | ----------- |
| padding     | `'small' \| 'medium' \| 'large' \| 'none'`        | The padding size of the card.    | `'large'`   |
| variant     | `'outline' \| 'filled' \| 'elevated' \| 'dashed'` | The variant of the card.         | `'outline'` |
| colorScheme | `'base' \| 'grey' \| 'purple' `                   | The color scheme of the card.    | `'base'`    |
| surface     | `'base' \| 'purple' `                             | The surface that the card is on. | `'base'`    |
| nested      | `boolean`                                         | Whether the card is nested.      | `false`     |
