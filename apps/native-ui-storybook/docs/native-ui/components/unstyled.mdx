# Using unstyled components

The `@utilitywarehouse/native-ui` library provides a set of unstyled components exported from `gluestack-ui` that
can be used to build components that have not yet been styled.

<NativeUIProvider>
  <Alert
    text="The unstyled components are not styled and should be used with caution. They are intended to be used to support engineers when a component isn't build yet so they can align with the gluestack-ui api."
    mb="$4"
  />
  <Alert
    colorScheme="warning"
    text="These compoenents will be relpaced in minor release versions and moved to the main library code."
    mb="$4"
  />
</NativeUIProvider>

## Usage

You can see a list of all the exported unstyled components [here](https://gluestack.io/ui/docs/components/typography/heading)
on the `gluestack-ui` documentation, see a full list below.

> The components should be wrapped in a `NativeUIProvider` component to ensure that the correct theme is applied.
