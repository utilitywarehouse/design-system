# Platform Specific

We can provide platform specific styles in the `sx` prop of the component. The `sx` prop takes an object with `_ios`, `_android` and `_web` specific styles.

<Canvas>
  <NativeUIProvider>
    <Center>
      <Box justifyContent="center" alignItems="center" h="100%">
        <Box alignItems="center">
          <Button $ios-bg="$red500" $web-bg="$green500" $android-bg="$gold500">
            <ButtonText>Button</ButtonText>
          </Button>
        </Box>
      </Box>
    </Center>
  </NativeUIProvider>
</Canvas>

```tsx
<Box justifyContent="center" alignItems="center" h="100%">
  <Box alignItems="center">
    <Button $ios-bg="$red500" $web-bg="$green500" $android-bg="$gold500">
      <ButtonText>Button</ButtonText>
    </Button>
  </Box>
</Box>

// or

<Button sx={{ _ios: { backgroundColor: '$red500' }, _web: { backgroundColor: '$green500' }, _android: { backgroundColor: '$gold500' } }}>
  <ButtonText>Button</ButtonText>
</Button>

// or

const StyledButton = styled(Button, {
  _ios: { backgroundColor: '$red500' },
  _web: { backgroundColor: '$green500' },
  _android: { backgroundColor: '$gold500' },
});
```
