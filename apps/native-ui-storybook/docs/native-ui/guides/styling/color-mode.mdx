# Color Mode

We can apply different styles based on different color mode by defining `_dark` and `_light` property in `sx` prop.

In the below example we are defining styles to be applied in dark mode in `_dark` object and for light mode in `_light` object.

<Canvas>
  <NativeUIProvider>
    <Center>
      <Button $dark-bg="$darkCyan600" $light-bg="$red600" p="$2">
        <ButtonText color="$white">Hello world</ButtonText>
      </Button>
    </Center>
  </NativeUIProvider>
</Canvas>

```tsx
<Button $dark-bg="$cyan600" $light-bg="$red600" p="$2">
  <ButtonText>Hello world</ButtonText>
</Button>

// or

<Button sx={{ backgroundColor: '$red600', p: "$2", _dark: { backgroundColor: '$darkCyan600' }, }}>
  <ButtonText>Hello world</ButtonText>
</Button>

// or

const StyledButton = styled(Button, {
  backgroundColor: '$red600',
  p: "$2",
  _dark: { backgroundColor: '$darkCyan600' },
});
```
