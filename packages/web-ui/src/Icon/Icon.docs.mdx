# Icon

Icon will render the given icon, and takes a color value from the [color design tokens](https://github.com/utilitywarehouse/customer-web-ui/blob/main/packages/design-tokens/src/colors.ts).

```tsx
import { Icon } from '@utilitywarehouse/web-ui';
import SuccessOutlinedIcon from '@utilitywarehouse/customer-ui-react-icons/24x24/SuccessOutlined';

const MyIcon = () => {
  return <Icon icon={SuccessOutlinedIcon} color="jewel" iconProps={{ width: 32, height: 32 }} />;
};
```

## Alternatives

You can always render the imported icon itself. The difference is that we
currently aren't able to pass the ref down to the SVG element, due to the
implementation in the react-icons package so in the Icon component we pass it
to a wrapper instead. You will also need to pass a color value to the `fill` prop
to change the icon color.

```tsx
import SuccessOutlinedIcon from '@utilitywarehouse/customer-ui-react-icons/24x24/SuccessOutlined';
import { colors } from '@utilitywarehouse/customer-ui-design-tokens';

const MyIcon = () => {
  return <SuccessOutlinedIcon fill={colors.jewel} width={32} height={32} />;
};
```
