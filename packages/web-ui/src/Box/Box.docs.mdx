# Box

Box is a low-level primitive, which supports theme-aware styling props, and can
be used for building any styled element.

* [Semantic HTML](#semantic-html)
* [Background](#background)
* [Props](#props)

## Semantic HTML

By default Box renders a div element, this can be customised using the
`component` prop.

```tsx
<Box component="span">
  ...
</Box>
```

## Background

Additionally to the system properties, Box supports pre-defined backgrounds
that provide visual context to child components. This means that components
will change foreground colour depending on the parent background colour.

```tsx
<Box background="lightTint">
  <Text variant="body" component="span">Dark text on neutral background</Text>
</Box>

<Box background="purple">
  <Text variant="body" component="span">Light text on inverse background</Text>
</Box>
```
