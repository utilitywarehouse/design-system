# Grid

A responsive layout grid adapts to screen size and orientation, ensuring
consistency across layouts.

This component is mostly based on the one from the '@mui/material' package, the
differences are documented below. Please
[check the MUI site for further documentation](https://mui.com/material-ui/react-grid/).

<Canvas of={Stories.GridStory} />

## Breakpoint Props

Instead of the stock MUI breakpoint prop names, the Web UI grid has props which
reflect our custom breakpoint names: {Object.keys(breakpoints).join(', ')}

## Default Columns

<Stack spacing={1} component="ul" padding={0} margin={0} width={200} divider={<Divider />}>
  {Object.keys(DEFAULT_COLUMNS).map(c => (

      <Text variant="body" component="span">

## Default Spacing

<Stack spacing={1} component="ul" padding={0} margin={0} width={200} divider={<Divider />}>
  {Object.keys(DEFAULT_SPACING).map(s => (

        {s}
      </Text>
