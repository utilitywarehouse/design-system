## Semantic HTML

By default `Background` renders a `div` element, this can be customised using the
`component` prop.

```tsx dark
<Background component="span">...</Box>
```

## Background colours

Additionally to the system properties, Background supports pre-defined background colours
that provide visual context to child components. This means that components
will change foreground colour depending on the parent background colour.
