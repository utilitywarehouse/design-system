# Text

Text renders the secondary UW font, Work Sans, to be used for body copy.

The system props are not available on the Text component, if you need custom typography then use the [Typography](?path=/docs/components-typography--documentation) component.
If necessary you can use `sx` as an escape hatch for one-off custom styling.

<Canvas of={Stories.TextVariants} />

## Color

You can pass any valid string color to the `Text` component. It is recommended
you use the `@utilitywarehouse/colour-system` package, and pass color values
from that into the component. The default color is `colorsCommon.brandMidnight`
from this package.

```tsx
import { colorsCommon } from '@utilitywarehouse/colour-system'

<Text color={colorsCommon.brandPrimaryPurple}>{...}</Text>
```
